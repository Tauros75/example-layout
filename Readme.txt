This program is an example of landing page of a website.The template provided 
by https://www.frontendmentor.io/ as a challenge. 

To run the program, in VSCode :
1. Direct to directory Layout_Example. 
2. If not installed yet, one must install "Live Sass Compiler" extension in VSCode
3. After installing Live Sass, click on "Watch Sass" 
4. Then click on "Go Live" 
5. The landing page of website will be shown in your browser
6. You can inspect the elements and etc by right click >> select inspect
